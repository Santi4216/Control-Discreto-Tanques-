# 1пёҸвғЈ CONEXIONES GPIO DEL ESP32-S3

## рҹ“Ҡ Tabla RГЎpida

| рҹ”Ң Pin | рҹ“Қ FunciГіn | рҹ”Ӣ TensiГіn | рҹ“Ө Salida | рҹ“қ Notas |
|:---:|:---|:---:|:---:|:---|
| **GPIO 17** | рҹ”· BOMBA PWM | 3.3V | вҶҗ L298N IN1 | Velocidad 0-255 |
| **GPIO 18** | рҹҹў SERVO 1 PWM | 3.3V | вҶҗ VГЎlvula T1 | 45В°=Cierra, 135В°=Abre |
| **GPIO 19** | рҹ”ө SERVO 2 PWM | 3.3V | вҶҗ VГЎlvula T2 | 45В°=Cierra, 135В°=Abre |
| **GPIO 7** | рҹ’Ў LED STATUS | 3.3V | вҶ’ LED Indicador | HIGH=ON, LOW=OFF |
| **GND** | рҹ”ұ TIERRA COMГҡN | 0V | вҶ” TODO | вҡ пёҸ **CRГҚTICO** |
| **5V** | вҡЎ ALIMENTACIГ“N | +5V | вҶ’ Servos | USB o externo |

# вқҢ TROUBLESHOOTING RГҒPIDO

| рҹ”ҙ PROBLEMA | рҹ”Қ CAUSA | вң… SOLUCIГ“N |
|:---|:---|:---|
| **LED no enciende** | USB no conectado | Conecta USB-C al PC |
| **Bomba no se mueve** | Tierra no compartida | Verifica GND comГәn |
| | GPIO 17 no conectado | Revisa L298N IN1 |
| | L298N sin potencia | Verifica +12V y GND |
| **Servo tiembla** | Ruido en seГұal | Separa cables potencia/seГұal |
| | Servo sin 5V | Verifica 5V en servo |
| **Servo no responde** | PWM invertido | Invierte ГЎngulos (45вҶ”135) |
| | GPIO 18/19 mal | Revisa conexiГіn a servos |
| **No hay comunicaciГіn** | Puerto COM incorrecto | Abre Arduino IDE, identifica COM |
| | Baud rate incorrecto | Verifica 115200 en cГіdigo |
| | Firmware no cargado | Sube firmware via Arduino IDE |
| **Fusible quema** | Cortocircuito en bomba | Desconecta y revisa bomba |
| | Polaridad invertida | Verifica +/- en L298N |
| **Todo funciona pero lento** | PWM bajo | Aumenta valor PWM (0-255) |
