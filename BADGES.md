## 💡 Personalización de Badges

### Cambiar colores:

| Color | Código |
|-------|--------|
| Azul | `blue` |
| Verde | `green` |
| Rojo | `red` |
| Naranja | `orange` |
| Gris | `lightgrey` |
| Éxito | `success` o `brightgreen` |
| Crítico | `critical` |
| Inactivo | `inactive` |

Ejemplo: `?style=flat-square&color=brightgreen`

### Cambiar estilos:

- `flat` - Plano
- `flat-square` - Cuadrado plano ⭐ Recomendado
- `plastic` - Plástico
- `for-the-badge` - Grande y vistoso ⭐ Recomendado
